💼 Job Portal Web Application

A web-based Job Portal System developed using PHP and MySQL as a college academic project.
This platform connects Job Seekers and Recruiters, allowing recruiters to post jobs and candidates to search and apply for opportunities.

🚀 Features

🔐 User Registration 

👨‍💼 Recruiter Job Posting

🔍 Job Search & Filtering

📄 Resume / Profile Management

📌 Apply for Jobs

🗂️ CRUD Operations (Create, Read, Update, Delete)

🔒 Session Management

🛠️ Structured Database Design

🛠️ Tech Stack
